# Changelog

All notable changes to this project will be documented here.

## Unreleased

- Made complete inspection of every readable body unit a hard prerequisite for whole-book structure, problem awareness, core claims, and argument-route output; extraction and sampling no longer qualify.
- Replaced the six-section provisional map with an eight-section whole-book analysis produced only after complete body inspection.
- Expanded Phase 2 from claim verification to critical positioning and verification, with user-selected perspective expansion before outside research.
- Expanded Phase 3 into integration, judgment, and transfer across writing, daily observation, work and relationships, self-reflection, and connections with other books.
- Added a perspective-and-transfer protocol, hid routine phase-status templates, and added new behavioral acceptance cases that remain pending live validation until independently run.
- Completed a three-phase reading model: source reconstruction, critical positioning and verification, and integration, judgment, and transfer.
- Defined phase entry, exit, evidence, browsing, interaction, and completion rules while keeping reading modes independent from phases.
- Separated evidence-phase progress from the reader's understanding status and added scoped provisional integration.
- Added notes-first navigation for book-plus-notes and notes-only review without treating paraphrases as source proof.
- Added Phase 2, Phase 3, partial-material, and notes-only behavior cases plus repository checks for the complete model.
- Prepared a public, cross-agent repository wrapper around the validated personal Skill.
- Added Chinese and English READMEs, MIT licensing, a reproducible demo GIF, public eval cases, CI, local installation evidence, and a Claude marketplace manifest.
- Replaced private-runtime assumptions with portable dependency guidance.
- Added a release validator that rejects private paths, credential-like values, missing public assets, invalid marketplace paths, and unreplaced repository-owner placeholders.

No public version or tag has been released yet.
